# Phase quick-260612-94j Plan 01: Styling Follow-up (Eternities Default, Gradient Buttons/Titles) Summary

**One-liner:** Default theme flipped to Blind Eternities (purple), gradient applied to all `default` buttons and 22 page title headings via new `.title-gradient` CSS utility aliased from `.hero-h1-gradient`.

## What Was Done

### Task 1 — Flip default theme to Eternities + gradient default button (cd2d5a5)

Three minimal edits:

1. **layout.tsx**: `data-theme="atlas"` changed to `data-theme="eternities"` — prevents first-paint flash for the new default.

2. **app-store.ts**: `uiTheme` initial state changed from `'atlas'` to `'eternities'` — new users (no persisted storage) load Eternities. Existing users with persisted `uiTheme: 'atlas'` continue on Atlas as expected.

3. **button.tsx**: `default` variant updated from flat `bg-primary` to theme-aware gradient + border + subtle inset highlight. No outer glow (preserved exclusive to `cta`).

   Old: `bg-primary text-primary-foreground [a]:hover:bg-primary/80`
   New: `bg-[linear-gradient(180deg,var(--gradient-cta-from),var(--gradient-cta-to))] text-[var(--color-cta-text)] border border-[var(--cta-border)] font-medium shadow-[inset_0_1px_0_rgba(255,255,255,0.25)] hover:opacity-95`

**Decision recorded:** `default` = gradient + border + `inset_0_1px_0_rgba(255,255,255,0.25)` inset; `cta` retains `shadow-[0_4px_28px_rgba(212,166,74,0.35),...]` outer glow. This preserves visual hierarchy — CTA is the louder hero button, default buttons work in dense UI without glow noise.

### Task 2 — Add .title-gradient utility and apply to all page H1s (d892a5e)

1. **globals.css**: Added `.title-gradient` as an alias to `.hero-h1-gradient` by extending all selector lists. Four new rules:
   - `.hero-h1-gradient, .title-gradient { ... }` (Atlas dark default)
   - `html[data-theme="eternities"] .hero-h1-gradient, html[data-theme="eternities"] .title-gradient { ... }` (Eternities dark)
   - `html.light .hero-h1-gradient, html.light .title-gradient { ... }` — light-mode override using `var(--gold)` to `var(--gold-deep)` (no washed-out tints on light bg)
   - `html.light[data-theme="eternities"] .hero-h1-gradient, html.light[data-theme="eternities"] .title-gradient { ... }` — `var(--accent)` to `var(--accent-deep)` for Eternities light

2. **22 page.tsx files** updated: added `title-gradient` to the main page H1, removed `text-[var(--color-accent)]`, `text-[var(--color-text)]`, `text-white`, `text-glow-purple`, `text-glow-gold` and inline `color:` styles from that element only.
