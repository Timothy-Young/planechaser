Create src/app/api/scryfall/planes/route.ts as a Next.js Route Handler (no 'use client'):

The handler exports an async GET function that:
1. Creates a Supabase server client using `createClient()` from `@/lib/supabase/server`
2. Queries card_cache for `card_type = 'planes'` — selects raw_json and cached_at
3. If a cache row exists AND it is less than 30 days old (30 * 24 * 60 * 60 * 1000 ms), return `Response.json(cached.raw_json)` immediately
4. If no cache or cache is stale: call `fetchPlanechaseCorpus()` from `@/lib/scryfall/client`
5. Upsert the result into card_cache with `{ card_type: 'planes', raw_json: planes, cached_at: new Date().toISOString() }`
6. Return `Response.json(planes)`
7. On any error from Scryfall fetch: return `Response.json({ error: 'Scryfall unavailable' }, { status: 503 })`

CRITICAL: This route handler fetches card METADATA (JSON) from Scryfall's API — that is permitted and intended. The prohibition from PITFALLS.md is against fetching and re-serving Scryfall IMAGE BYTES. This route never fetches from cards.scryfall.io — only from api.scryfall.com.

After cache is populated, response time must be <100ms (Supabase read, no Scryfall API call).

Create src/hooks/usePlaneCorpus.ts as a 'use client' TanStack Query hook:

  'use client'
  import { useQuery } from '@tanstack/react-query'
  import type { PlaneCorpusCard } from '@/lib/scryfall/types'

  export function usePlaneCorpus() {
    return useQuery<PlaneCorpusCard[]>({
      queryKey: ['plane-corpus'],
      queryFn: async () => {
        const res = await fetch('/api/scryfall/planes')
        if (!res.ok) throw new Error('Scryfall unavailable')
        return res.json()
      },
      staleTime: 30 * 24 * 60 * 60 * 1000,  // 30 days — matches cache TTL
      gcTime: 30 * 24 * 60 * 60 * 1000,
      retry: 2,
    })
  }

The hook returns `{ data: PlaneCorpusCard[] | undefined, isLoading, isError, error }` — callers handle loading/error states.
