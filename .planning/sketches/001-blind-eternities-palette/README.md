# Sketch 001: Blind Eternities Palette

## Design Question
Should PlaneChaser shift off stock Tailwind violet/amber to a bespoke palette, replace the blurred-orb hero with MTG-native geometry, and standardize a 48px+ gradient CTA button — and which direction feels right?

## How to View
open .planning/sketches/001-blind-eternities-palette/index.html

## Variants
- **A: Current (baseline)** — exact reproduction of today's styling: violet-600/purple-400/amber-500, blurred orbs, sparkles pill badge, 32px secondary buttons.
- **B: Blind Eternities** — deeper void-indigo background (#0a0813), hue-shifted purple (#b18aff), warm parchment-gold (#d9a648), purple-tinted grays instead of slate, hedron-constellation hero, gold-ruled badge, gradient-text H1, 48–52px buttons.
- **C: Planar Atlas** — gold-forward identity: gold gradient H1 and CTA, warm parchment text (#f5f1e8), portal-ring hero geometry, gilded plane frame. Purple demoted to secondary.

## What to Look For
- Does B still feel like "your app," just sharper? (That's the goal — B is the recommended direction.)
- Hero background: hedron constellation (B) / portal rings (C) vs blurred orbs (A) — which feels MTG vs generic AI?
- Button row in the in-game strip: 44–52px targets (B/C) vs 32px (A) on the 375px viewport (use the toolbar).
- Text tint: B's purple-gray secondary text vs A's slate — subtle but changes the whole temperature.
- C tests whether gold-primary is exciting or too "fantasy parchment."
