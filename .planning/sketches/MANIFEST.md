# Sketch Manifest

## Design Direction
PlaneChaser's existing identity (dark multiverse, Cinzel + Inter, purple/gold, glassmorphism + glow) is right and stays. The goal is a de-genericize pass: move the palette off stock Tailwind hex values to bespoke tints, replace AI-cliché hero patterns (blurred orbs, sparkles pill) with MTG-native geometry (hedrons, planar portals), and systematize buttons with proper mobile touch targets (≥44px) and a first-class gradient CTA variant.

## Reference Points
- MTG Planechase card frames (gold-ruled borders, engraved serif nameplates)
- Zendikar hedron iconography
- The current production app at planechaser/src — variant A in sketch 001 is the baseline
